## 3. Seccion de Contexto (Educativa)

### Titulo
```
Entendiendo el Sistema de Pensiones en Mexico
Una breve historia y lo que significa para ti
```

### Timeline

**1973 - Ley del Seguro Social**
```
Pension definida basada en semanas y salario.
Si empezaste a cotizar antes de julio 1997, estas aqui.
```

**1997 - Sistema AFOREs**
```
Cuentas individuales. Tu pension depende de lo que ahorres.
La mayoria de trabajadores actuales.
```

**2020 - Reforma de Pensiones**
```
Reduce requisito a 1,000 semanas.
Aumenta aportaciones patronales gradualmente hasta 2030.
```

**2024 - Fondo de Pensiones para el Bienestar**
```
Complementa pensiones hasta el 100% del ultimo salario.
Programa nuevo, sostenibilidad por confirmar.
```

### Tabla Comparativa

| Sistema | Lo Bueno | Lo Incierto |
|---------|----------|-------------|
| Ley 73 | Pension definida, certeza, generalmente mejor | Solo para quienes empezaron antes de julio 1997 |
| Ley 97 (AFORE) | Tu dinero, transparencia, portabilidad | Pensiones bajas (15-35% del salario tipicamente) |
| Fondo Bienestar | Puede complementar hasta 100% del salario | Programa nuevo (2024), sostenibilidad incierta |

### Framework de Control

**VERDE - Tu controlas:**
- Aportaciones voluntarias
- Eleccion de AFORE
- Edad de retiro
- Continuidad laboral

**AMARILLO - Influencia parcial:**
- Rendimientos AFORE
- Salario futuro
- Semanas cotizadas

**ROJO - No controlas:**
- Futuro del Fondo Bienestar
- Cambios en leyes
- Esperanza de vida
- Inflacion

- Cambiar a mejor AFORE
- Retrasar retiro si es posible

## 12. Glosario (Terminos Clave)

### UMA
```
Unidad de Medida y Actualizacion. Indicador que reemplaza al
salario minimo para calculos oficiales desde 2016.
```

### SBC
```
Salario Base de Cotizacion. El salario registrado ante el IMSS,
que puede ser diferente al salario real.
```

### Semanas cotizadas
```
Numero de semanas que tu empleador ha reportado al IMSS.
Cada semana suma a tu historial.
```

### Cesantia
```
Pension anticipada entre los 60 y 64 anos. Tiene un factor de
reduccion (75% a los 60, hasta 95% a los 64).
```

### Vejez
```
Pension completa a partir de los 65 anos, sin factor de reduccion.
```

### Retiro programado
```
Modalidad de pension donde tu AFORE te paga mes a mes hasta
agotar tu saldo. El monto se recalcula cada ano.
```

### Renta vitalicia
```
Modalidad de pension donde una aseguradora te paga de por vida
a cambio de tu saldo AFORE.
```

### Densidad de cotizacion
```
Porcentaje de tu vida laboral en que has cotizado.
Promedio nacional: ~50-65%.
```

### Fondo Bienestar
```
Fondo de Pensiones para el Bienestar (2024). Complementa pensiones
de trabajadores con salario <= $17,364/mes hasta el 100% de su
ultimo salario.
```

### Modalidad 40
```
Continuacion voluntaria en el regimen obligatorio. Permite a
ex-trabajadores seguir cotizando para mejorar su pension Ley 73.
```

### IRN
```
Indicador de Rendimiento Neto. Mide el rendimiento de una AFORE
descontando comisiones. Mayor IRN = mejor desempeno.
```

### Ley 73
```
Sistema de pension anterior a julio 1997. Pension definida basada
en salario y semanas. Generalmente mas favorable.
```

### Ley 97
```
Sistema de pension actual (AFORE). Cuenta individual donde cada
quien ahorra para su propia pension.
```
